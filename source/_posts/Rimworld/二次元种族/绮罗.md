你说这未开启基因的全局学习系数是90%,开启了是100%×0.9还是90%,岂不是一样?实际是不一样的,当遇到一些加减算的时候就能明显体现出来,比方说神经超频+25%全局学习系数,在前者的最终值为90%+25%=115%,后者则是(100%+25%)×0.9=112.5%,直接差出来了2.5%,这样的乘算会将之后的修正值也进行缩小

绮罗族仅可穿戴绮罗衣物(设置里可改),且必定不会亲戚生成器.

闪避率是绮罗的核心系统,绮罗族的近战闪避率上限为99%,而不是原版的50%.同时,绮罗族的近战闪避率最终结算时额外乘以(1+4.9%×格斗等级)(最大1.98),例如,一个10级格斗的绮罗,其先乘以1.2的基因闪避率,折合12点,进行后期处理曲线,变为14%,最后再变为 14%×(1+4.9%×10)=20.86%  
同时,绮罗族拥有独特的"远程闪避率"属性,一般为近战闪避率的1/3,可由装备进一步加成,上限99%

### 毒素

中世纪的远程武器可使用"麻痹毒瓶"或"致命毒瓶"来为武器"附魔"毒伤.每点附魔的实际伤害会增加0.3%×(1-毒素抗性)×(1/体型)的相应毒素.

当被麻痹毒瓶攻击时,获得"麻痹毒"状态,此状态每8h增长100%,1到1.5天后消失.
 -  1% 意识-10% 移动能力-10%
 -  10% 意识最大80% 移动操作能力最大65% 语言视觉能力最大90% 消化能力85%
 -  20% 意识最大60% 移动能力最大25% 操作能力最大35% 语言视觉能力最大70% 消化能力最大65%
 -  25% 意识最大10%
 -  90% 意识最大75% 移动能力-20% 操作能力-30% 语言视觉能力-15% 消化能力-20%

我很难说这个毒有没有用,数值给的非常原版,靠伤害叠到25%锁意识需要83.3伤害,dpsa最高的传奇绮罗猎人短弓在100%瞄准时间下为12.9,在50%瞄准时间下为18,五把弓就能一秒干倒一个,实现全自动割腰子  
注意到每8h增长100%,再注意到25%时就直接意识10%强制倒地了,所以随便找个毒素抗性小于1的,叠一下毒然后溜俩小时也能倒地,当然这和放血溜人到晕倒没啥本质区别  
同时这玩意还吃毒素抗性和体型,这个机制变得更弱了,或许可以拿来打一些boss,见仁见智吧.

**注意,商户系统大量使用了AI扒取源码来得到细节内容,仅供参考,本人并不懂代码,所以未做验证,仅作参考**

绮罗事件扩展添加了独特的"驻殖民地商人"机制,当与绮罗为盟友时,可以前往8半径内的绮罗派系进行调查招募
 -  刷新:第一次调查/招募后开始计算商人刷新,每20~30天刷新一次商人名单,刷新后需重新调查.每次刷新1~4个商人,按权重独立抽取,可重复
 -  调查商户花费300银(显示400银,但实际扣费300银),注意商户已经是被决定了的,调查只是显示出来,所以sl并没有卵用
 -  招募共有三种方式,具体为:
     -  张贴广告:100银,6天cd,有$70\%\times f(智识+艺术)$概率随机招募1名商户,若是绮罗族来招募,额外加5%成功率
     -  上门拜访:300银,10天cd,有$80\%\times f(谈判能力)$概率招募指定1名商户,若是绮罗族来招募,额外加5%成功率
     -  集会宣讲:600银,10天cd,有$90\%\times f(智识+社交)$概率随机招募1名商户,并有前者概率的1/3随机招募1名商户,两者独立判定,并不是只有前者才能出现后者.若是绮罗族来招募,额外加5%成功率
     -  特别地,如果招募成功,你会看到商人名单里已经少人了,可以快速判断是否招募成功
     -  <details>
        <summary>点击查看线性插值函数</summary>

        $f(智识+艺术)$和$f(智识+社交)$的线性插值都是$6\rightarrow 0 ; 10\rightarrow 0.4 ; 20\rightarrow 0.7 ;30\rightarrow 0.8 ; 40\rightarrow 0.9$  
        $f(谈判能力)$的线性插值为$0.4\rightarrow 0 ; 0.6\rightarrow 0.4 ; 0.8\rightarrow 0.6 ; 1.5\rightarrow 0.85 ; 2\rightarrow 0.9$  
        叽里咕噜说什么呢,反正你拿智识+社交最高的去集会宣讲,谈判能力最高的去上门拜访就完事了(另外有点搞的是绮罗谈判能力天生就只有80%,即使是20级社交,也仅有152%谈判能力,穿了单片眼镜也仅有161.5%)
        </details>
 -  除主动招募外,还有流浪绮罗商人(`Kiiro_WanderingMerchant`)任务,权重为1,再次触发最小间隔160天,仅当与绮罗结盟且地图在地面时可触发.会按照权重招募一名商户  
    另外勇者开局会在15~18天固定刷一次该任务
