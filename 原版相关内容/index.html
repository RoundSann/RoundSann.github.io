工作速度计算

符号定义:

  • 我们记kkk表示一项工作速度的基础值,例如智人种的普通工作速度基础值为100%
  • 我们记GGG表示全局工作速度
  • 我们定义算符: (a′)x(bM):=(1+a(x−1))(最大取b)(a')x(bM):=(1+a(x-1))(最大取b)(a′)x(bM):=(1+a(x−1))(最大取b).
    对应游戏中的 权重a,最大值b. 在表格中,我们直接简记为a(bM)
  • 我们定义算符: (′c)y(dM):=(cy−c)(最大取d)('c)y(dM):=(cy-c)(最大取d)(′c)y(dM):=(cy−c)(最大取d).
    对应游戏中的 影响c,最大值(d/c +1). 在表格中,我们直接简记为c(dM)
  • 部分社交属性的权重带有豁免线.其会降低能力值过低对属性造成的影响
    具体为:当能力值<豁免线时,则使用 能力值/豁免线 代替能力值进行计算
    对应游戏中的 权重a,最大值b,如果低于豁免线. 在表格中,我们直接简记为(豁免线)a(bM)
    例如,教化能力的听觉权重为 (80%)0.9(1M),则对于30%听觉能力的人来说,其听觉对教化能力的实际影响为(1+0.9(0.3/0.8−1))(1M)=0.4375(1+0.9(0.3/0.8-1))(1M)=0.4375(1+0.9(0.3/0.8−1))(1M)=0.4375,从而30%听觉能力对教化能力造成总共42.75%影响.

衣物护甲

衣物护甲的计算方式为:

(基础值+材料防护能力×材料效果乘数)×品质偏移(基础值+材料防护能力\times 材料效果乘数)\times 品质偏移(基础值+材料防护能力×材料效果乘数)×品质偏移

其中,材料防护能力指材料的防御数值,材料效果乘数则是服装自带的属性,代表能把材料的防御数值发挥多少

材料整体分为金属/皮革/纤维三类.
金属防护能力较低,但金属制品一般本身材料效果乘数高.通常在90%
皮革或纤维自身防护能力高,但皮革纤维制品一般本身材料效果乘数低,通常在30%

例如:

  • 原版玻璃钢板甲的利防为 114%×90%=102114\%\times 90\%=102%114%×90%=102.若为传奇品质,则为102%×1.8=183.6%102\%\times 1.8=183.6\%102%×1.8=183.6%
  • 原版敲击兽大衣的利防为 208%×30%=62.4%208\%\times 30\%=62.4\%208%×30%=62.4%.若为传奇品质,则为62.4%×1.8=112.32%62.4\%\times 1.8=112.32\%62.4%×1.8=112.32%

割伤(Cut)

割伤是最常见的近战伤害类型.造成割伤时,有概率造成伤害分裂,将伤害增强并平均到多个部位.具体如下:

  • 有30%概率不分裂伤害,直接对部位造成伤害
  • 有45%概率将伤害分裂成两份,对每个部位造成70.6%伤害,总计141%
  • 有20%概率将伤害分裂成三份,对每个部位造成54.5%伤害,总计164%
  • 有5%概率将伤害分裂成四份,对每个部位造成44.4%伤害,总计178%

受伤流程如下:

  1. 选中一个外部器官作为初始受击器官,并对初始受击器官进行护甲减伤判定
  2. 根据概率决定分裂次数.30%概率不分裂,45%概率分裂为两份,20%概率分裂成三份,5%概率分裂成四份
  3. 在以下列表中随机选择(分裂份数-1)个器官,将其加入受击器官
    • 初始受击器官的子器官(例如足的子器官趾)
    • 初始受击器官的父器官(例如足的父器官腿)
    • 初始受击器官的父器官的子器官,不包括自己(例如足的父器官腿的子器官股骨)
  4. 对每个(初始)受击器官分别造成 2.4×伤害/(分裂份数+1.4)2.4\times 伤害/(分裂份数+1.4)2.4×伤害/(分裂份数+1.4) 伤害,总计 2.4×伤害×分裂份数/(分裂份数+1.4)2.4\times 伤害\times 分裂份数/(分裂份数+1.4)2.4×伤害×分裂份数/(分裂份数+1.4) 伤害

注意,由于护甲减伤判定是先于器官选择的,所以可以"逃护甲关税",例如,割伤击中了足,而众所周知rimworld的动力甲是不包括足部和手部的,假如此次割伤分裂到了足的父器官-腿上,那么伤害会绕过腿的护甲判定,直接对腿造成伤害,从而逃避了护甲.同样,也可能导致击中腿蔓延到足,导致足也减伤.

刺伤(Stab)

刺伤分为近战刺伤和远程刺伤两种,其都有将伤害从内部器官向外部延伸的特性:

  • 造成近战刺伤时,有60%概率将40%伤害传递到内部的随机子器官,并将外部伤害衰减为原来的75%,造成总共115%伤害.
  • 造成远程刺伤时,若击中外部器官,则有60%概率将40%伤害传递到内部的随机子器官,并将外部伤害衰减为原来的75%,造成总共115%伤害
  • 造成远程刺伤时,若击中内部器官,则沿着器官链一路向上传递到第一个外部器官,内部器官造成40%伤害,外部器官造成75%伤害,造成总共75%+40%x75\%+40\%x75%+40%x伤害,xxx为内部器官个数

以下内容是AI扒的,不一定正确仅供参考.我仅验证了远程刺伤能够造成二次穿透,近战暂时未发现二次穿透
受伤流程如下:

  1. 选取初始受击器官:
    • 如果是近战刺伤,则选中一个外部器官作为受击器官
    • 如果是远程刺伤,则选中任意器官作为受击器官
  2. 若该器官是外部器官,则有60%概率选取其中一个内部子器官代替当前器官作为受击器官
  3. 对受击器官进行护甲减伤判定
  4. 沿着受击器官的父器官一路传递直到第一个外部器官,对所有的内部器官造成40%伤害,对所有的外部器官造成75%伤害

例如,某器官树为:A(外部)-B(内部)-C(外部)-D(内部)-E(内部),无护甲,伤害100

  • 若为近战刺伤,则初始选中只能选中A/C(近战仅能选中外部器官),同时有60%概率传递到B/D(内部子器官),若传递,则A/C受到75伤害,B/D受到40伤害,总伤害115
  • 若为远程刺伤,则ABCDE都可作为初始选中(任意器官)
    1. 若选中A/C,则其同上面的近战,至多只能造成115伤害
    2. 当选中B/D时,由于B/D是内部器官,无法二次选中内部子器官,直接开始一路传递,造成115伤害
    3. 当选中E时,由于E是内部器官,同样开始一路传递直到第一个外部器官C,即C受到75伤害,D,E受到40伤害,总伤害155
  • 注:在此例中,向上传递到C时已经是外部器官了,所以AB不会因BCD的传递而受伤

从此例子中我们可以发现,近战刺伤有且仅能传递一次,总期望40%+60%×115%=109%40\%+60\%\times 115\%=109\%40%+60%×115%=109%.而远程刺伤只要脸够白且器官树够长,可以一路传递上去造成较高实际伤害.
另外值得一提的是原版远程刺伤只有重标枪(25伤10穿),脊刺(30伤100穿)和骨刺射击(单爪兽7伤100穿).原版可造成两次传递的内部器官树有且仅有 头(外部)-颅骨-大脑

钝伤(Blunt)

钝伤除了众所周知的会把伤害延伸到父器官以外,还有40%概率吸走外部的10%~20%伤害,并对随机子器官造成10%~20%+20%~35%伤害

受击流程如下:

  1. 选中一个外部器官作为受击器官,并对初始受击器官进行护甲减伤判定
  2. 有40%概率触发内部打击,将伤害降低为原始伤害的80%~90%
  3. 将受击器官造成伤害,若伤害溢出直接肘飞器官了,则将溢出的伤害攻击到受击器官的父器官上,如此往复,直到无溢出伤害或无父器官
  4. 若触发了内部打击,对受击器官的子器官造成10%~20%(在2中被抽走的伤害)+20%~35%的伤害
  5. 若击中躯干,有较低概率造成2s眩晕,若击中头,有较高概率造成2s眩晕,若打中手臂或腿,不造成眩晕

注意,由于有的外部器官是没有子器官的,当选中了没有子器官的外部器官,同时触发了内部打击时,被吸走给内部器官的10%~20%伤害就被直接吞掉消失了

炸伤(Bomb)

受到炸伤时,有概率使伤害翻倍,具体如下:

  1. 随机选择1~4个部位受伤
  2. 将伤害均分到这些部位,并分别计算护甲减伤
  3. 若受伤部位存在"父器官",则将伤害同时延续到父器官上

抓伤(Scratch)

动物的伤害类型,与割伤类似,固定选双目标,分别67%伤害,总共134%伤害

以下内容是AI扒的,不一定正确仅供参考.全部未经验证
受伤流程如下:

  1. 选中一个外部器官作为初始受击器官,并对初始受击器官进行护甲减伤判定
  2. 在以下列表中随机选择1个外部器官,将其加入受击器官
    • 初始受击器官的子器官(例如足的子器官趾)
    • 初始受击器官的父器官(例如足的父器官腿)
    • 初始受击器官的父器官的子器官(例如足的父器官腿的子器官股骨)
    • 若以上三个都不存在,则仅造成一次100%伤害
  3. 对这两个受击器官分别67%伤害,总共134%伤害
